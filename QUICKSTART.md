## First Launch

When you first run BetterNotch, you'll see a small black bar at the top center of your screen where the notch is located. This is the collapsed state.

## Feature Guides

### Media Controls
1. Play music in any supported app (Spotify, Apple Music, etc.)
2. Hover over the notch and select the Media tab
3. You'll see album art, track info, and playback controls
4. Use the volume slider to adjust audio levels
5. Click previous/next to skip tracks

### Notifications
1. Open the Notifications tab
2. View all recent notifications in one place
3. Hover over a notification to see the dismiss button
4. Click the dismiss button to remove individual notifications

### Calendar
1. Grant calendar permissions when prompted
2. Open the Calendar tab to see today's events
3. Events show:
   - Time and duration
   - Event title and location
   - Status (happening now, upcoming, etc.)
4. Click the arrow icon on an event to open it in Calendar app
5. Use the refresh button to update events

### Apps Launcher
1. Open the Apps tab
2. Use the search bar to find specific apps
3. Click any app icon to launch it
4. Apps are automatically discovered from your system

## Troubleshooting

### Notch doesn't appear
- Make sure you have a MacBook with a notch (2021 or later)
- Check that BetterNotch is running (look for the menu bar icon)
- Try quitting and restarting the app

### Calendar events not showing
- Grant calendar permissions in System Settings > Privacy & Security > Calendars
- Click the refresh button in the Calendar tab
- Make sure you have events in your Calendar app

### Media controls not working
- Ensure media is actually playing in a supported app
- Grant necessary permissions if prompted
- Try playing/pausing from the original app first

### Files won't drop into tray
- Make sure you're dropping onto the tray area (it will highlight)
- Check file permissions in System Settings
- Try with a simple text file first
