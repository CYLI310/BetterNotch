## How It Works

1. **Hover to Expand**: Move your cursor over the notch to reveal the Dynamic Island
2. **Pin to Stay**: Click the pin icon to keep it expanded
3. **Switch Tabs**: Navigate between different features using the tab bar
4. **Interact**: Use all features with smooth, native macOS interactions

## Design Philosophy

BetterNotch brings the iOS Dynamic Island experience to macOS with:
- **Glassmorphism**: Beautiful frosted glass effects
- **Smooth Animations**: Fluid transitions and interactions
- **Native Integration**: Feels like a natural part of macOS
- **Dark Mode**: Designed for the modern macOS aesthetic

## Technical Details

- **Built with SwiftUI**: Modern, declarative UI framework
- **AppKit Integration**: Custom NSWindow for precise control
- **EventKit**: Native calendar integration
- **MediaPlayer**: System media control integration
- **Sandboxed**: Secure and privacy-focused

## Requirements

- macOS 14.0 or later
- MacBook Pro 14" or 16" (2021 or later) with notch
- Xcode 15.0 or later (for building from source)
